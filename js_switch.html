⚡ Switch операторы

Көп мәнді салыстырудың тиімді жолы

if...else if

Дәстүрлі тәсіл

let day = "дүйсенбі"; if (day === "дүйсенбі") { console.log("Апта басы"); } else if (day === "сейсенбі") { console.log("Екінші күн"); } else if (day === "сәрсенбі") { console.log("Орта күн"); } else if (day === "бейсенбі") { console.log("Төртінші күн"); } else if (day === "жұма") { console.log("Апта соңы"); } else { console.log("Демалыс"); }
Артықшылықтары:
✅ Күрделі шарттарға жарайды
✅ Әртүрлі операторлар пайдалануға болады
✅ Барлық жерде қолдауы бар
VS
switch

Заманауи тәсіл

let day = "дүйсенбі"; switch (day) { case "дүйсенбі": console.log("Апта басы"); break; case "сейсенбі": console.log("Екінші күн"); break; case "сәрсенбі": console.log("Орта күн"); break; case "бейсенбі": console.log("Төртінші күн"); break; case "жұма": console.log("Апта соңы"); break; default: console.log("Демалыс"); }
Артықшылықтары:
✅ Оқуға оңай
✅ Жылдам орындалады
✅ Қысқа және таза код

🏗️ Switch құрылымы

switch (expression)
⬇️
case value1:
// код
break;
⬇️
case value2:
// код
break;
⬇️
default:
// default код

📋 Негізгі ережелер:

  • switch - мәнді тексеретін кілт сөз
  • case - әр мүмкін мән үшін
  • break - кодтың орындалуын тоқтатады
  • default - ешқандай case сәйкес келмесе
  • Тек === (қатаң салыстыру) пайдаланады

🧮 Калькулятор функциясы

function calculate(operation, a, b) { switch (operation) { case "+": case "қосу": return a + b; case "-": case "азайту": return a - b; case "*": case "көбейту": return a * b; case "/": case "бөлу": if (b === 0) { return "Нөлге бөлуге болмайды!"; } return a / b; case "%": case "қалдық": return a % b; case "**": case "дәреже": return a ** b; default: return "Белгісіз операция: " + operation; } } // Мысалдар: console.log(calculate("+", 10, 5)); // 15 console.log(calculate("бөлу", 20, 4)); // 5

⚡ Өнімділік салыстыруы

if...else if
O(n)

Барлық шарттарды тізбектеп тексереді. Case көп болған сайын баяулайды.

switch
O(1)

Jump table пайдаланады. Case саны көп болса да жылдам жұмыс істейді.

Object lookup
O(1)

Объект қасиеттері арқылы тікелей қол жеткізу. Функциялармен жұмыс істеуге ыңғайлы.

📈 Қашан қайсысын пайдалану керек:

  • if...else if - күрделі шарттар, әртүрлі операторлар
  • switch - бір мәнді көп нұсқамен салыстыру
  • Object lookup - динамикалық мәндер, функциялар
