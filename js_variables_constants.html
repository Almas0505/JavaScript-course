Сабақ 2: Айнымалылар және деректер түрлері

📦
Айнымалы дегеніміз не?

Айнымалы - бұл ақпаратты сақтауға арналған контейнер. Қорапша сияқты, ішіне әртүрлі заттарды салып қоюға болады:

// Айнымалы жариялау (декларация) let myName = "Асем"; // Мәтін (string) let myAge = 25; // Сан (number) let isStudent = true; // Логикалық мән (boolean) // Айнымалыларды пайдалану console.log("Менің атым: " + myName); console.log("Менің жасым: " + myAge); console.log("Мен студентпін: " + isStudent);

🎮 Айнымалы құрушысы

let myName = "Алия";
🔤
Айнымалы жариялау тәсілдері

JavaScript-те айнымалы жариялаудың үш тәсілі бар:

Кілт сөз Сипаттамасы Өзгертуге болады Қайта жариялау
let Заманауи тәсіл, блок ауқымы ✅ Иә ❌ Жоқ
const Тұрақты мән, өзгермейді ❌ Жоқ ❌ Жоқ
var Ескі тәсіл, функция ауқымы ✅ Иә ✅ Иә
// let - мәнін өзгертуге болады let city = "Алматы"; city = "Астана"; // Дұрыс // const - тұрақты мән const country = "Қазақстан"; // country = "Франция"; // Қате! Өзгертуге болмайды // var - ескі тәсіл (қолданбаған дұрыс) var oldVariable = "Бұл ескі тәсіл";

JavaScript-тегі деректер түрлері

JavaScript-те жеті негізгі деректер түрі бар:

📝

String (Мәтін)

"Сәлем" немесе 'Әлем'

🔢

Number (Сан)

42, 3.14, -15

✅

Boolean (Логика)

true немесе false

❓

Undefined

Мәні жоқ

⭕

Null

Бос мән

📋

Object (Объект)

{name: "Айжан"}

🔣

Symbol

Бірегей белгі

🎯 Деректер түрін анықтау

Жоғарыдағы карточкаларды басып көріңіз

Түрлер арасында түрлендіру

JavaScript-те деректерді бір түрден екіншіге түрлендіруге болады:

// Санды мәтінге түрлендіру let num = 123; let str = String(num); // "123" let str2 = num.toString(); // "123" // Мәтінді санға түрлендіру let text = "456"; let number = Number(text); // 456 let number2 = parseInt(text); // 456 // Boolean-ға түрлендіру let bool1 = Boolean(1); // true let bool2 = Boolean(0); // false let bool3 = Boolean(""); // false let bool4 = Boolean("text"); // true
