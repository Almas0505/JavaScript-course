🎯 Кіріспе: Циклдар не үшін керек?

Деректер өңдеу

Массивтер мен объектілермен жұмыс

1️⃣ FOR циклі

FOR циклі - ең көп қолданылатын цикл түрі. Белгілі бір санда қайталау керек болғанда қолданылады. 

// FOR циклінің синтаксисі
for (бастау; шарт; өзгерту) {
    // орындалатын код
}

// Нақты мысал: 1-ден 5-ке дейін санау
for (let i = 1; i <= 5; i++) {
    console.log('Итерация ' + i);
}

// Массивпен жұмыс
const сандар = [10, 20, 30, 40, 50];
for (let i = 0; i < сандар.length; i++) {
    console.log(сандар[i]);
}

🎮 Интерактивті FOR циклі

Жылдамдық: 0.8s

2️⃣ WHILE циклі

WHILE циклі - шарт ақиқат болғанша орындалады. Қанша рет қайталанатыны алдын ала белгісіз болғанда қолданылады. 

// WHILE циклінің синтаксисі
while (шарт) {
    // орындалатын код
}

// Мысал: Кездейсоқ сан 7 болғанша
let кездейсоқСан = 0;
let әрекет = 0;

while (кездейсоқСан !== 7) {
    кездейсоқСан = Math.floor(Math.random() * 10) + 1;
    әрекет++;
    console.log('Әрекет ' + әрекет + ': ' + кездейсоқСан);
}
⚠️
Абай болыңыз! While циклінде шарт ешқашан жалған болмаса, шексіз цикл пайда болады. Әрқашан циклден шығу жолын қарастырыңыз!

🎲 WHILE циклі - "7 санын тап" ойыны

Компьютер 7 санын тапқанша кездейсоқ сандар генерациялайды:

3️⃣ DO...WHILE циклі

DO...WHILE циклі - кем дегенде бір рет орындалады, содан кейін шартты тексереді. 

// DO...WHILE циклінің синтаксисі
do {
    // орындалатын код (кем дегенде 1 рет)
} while (шарт);

// Мысал: Дұрыс жауап алғанша сұрау
let жауап;
do {
    жауап = prompt('Қазақстанның астанасы қай қала?');
} while (жауап !== 'Астана');

❓ DO...WHILE - Викторина

4️⃣ FOR...OF және FOR...IN циклдары 

// FOR...OF - массив элементтерінің мәндері
const жемістер = ['алма', 'банан', 'апельсин'];

for (const жеміс of жемістер) {
    console.log(жеміс);
}

// FOR...IN - объект қасиеттері
const адам = {
    аты: 'Айдос',
    жасы: 25,
    қаласы: 'Алматы'
};

for (const қасиет in адам) {
    console.log(қасиет + ': ' + адам[қасиет]);
}

🔍 Массивтер мен объектілермен жұмыс

5️⃣ BREAK және CONTINUE 

// BREAK - циклді толық тоқтатады
for (let i = 1; i <= 10; i++) {
    if (i === 6) {
        break; // 6-да тоқтайды
    }
    console.log(i);
}

// CONTINUE - келесі итерацияға өтеді
for (let i = 1; i <= 10; i++) {
    if (i % 2 === 0) {
        continue; // жұп сандарды өткізеді
    }
    console.log(i); // тек тақ сандар
}

🛑 Break және Continue көрсетілімі

6️⃣ Кірістірілген циклдар 

// Көбейту кестесі
for (let i = 1; i <= 3; i++) {
    for (let j = 1; j <= 3; j++) {
        console.log(i + ' × ' + j + ' = ' + (i * j));
    }
}

// Жұлдызша пирамида
for (let i = 1; i <= 5; i++) {
    let жол = '';
    for (let j = 1; j <= i; j++) {
        жол += '⭐ ';
    }
    console.log(жол);
}

⭐ Өрнектер салу

📊 Циклдарды салыстыру

Цикл түрі Қашан қолдану Ерекшелігі
FOR Қайталау саны белгілі болғанда Есептегіш, бастау және аяқтау нүктелері бар
WHILE Шарт бойынша қайталау керек болғанда Қайталау саны алдын ала белгісіз
DO...WHILE Кем дегенде 1 рет орындау керек болғанда Алдымен орындайды, содан кейін тексереді
FOR...OF Массив элементтерімен жұмыс Тек мәндерді алады, индекс керек емес
FOR...IN Объект қасиеттерімен жұмыс Қасиет атауларын алады

💪 Практикалық тапсырмалар

🧮 Тапсырма 1: Факториал есептеу

Факториал дегеніміз - 1-ден берілген санға дейінгі барлық сандардың көбейтіндісі. Мысалы: 5! = 1 × 2 × 3 × 4 × 5 = 120

🔢 Тапсырма 2: Жай сандарды табу

Жай сан - тек өзіне және 1-ге бөлінетін сан (2, 3, 5, 7, 11...)

🎮 Тапсырма 3: FizzBuzz ойыны

1-100 дейін санау: 3-ке бөлінсе "Fizz", 5-ке бөлінсе "Buzz", екеуіне де бөлінсе "FizzBuzz"
