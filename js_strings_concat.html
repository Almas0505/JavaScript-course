📝 Мәтіндер мен біріктіру

JavaScript-тегі мәтіндермен жұмыс істеу

💬
Мәтін жасау
// Үш тәсіл мәтін жасауға: let name1 = "Асем"; // Қос тырнақша let name2 = 'Арман'; // Жалғыз тырнақша let greeting = `Сәлем!`; // Backtick (template literal) console.log(name1, name2, greeting);
Ескерту: Тырнақшалар бірдей болуы керек! "Асем' - қате!
🔗
Біріктіру (+)
// + операторымен біріктіру let firstName = "Айжан"; let lastName = "Ахметова"; let fullName = firstName + " " + lastName; console.log(fullName); // "Айжан Ахметова" // Сандармен біріктіру let age = 25; let message = "Менің жасым: " + age; console.log(message); // "Менің жасым: 25"
📋
Template Literals
// Template literals - ең ыңғайлы тәсіл let name = "Берік"; let age = 30; let city = "Алматы"; let info = `Менің атым ${name}, жасым ${age}, ${city} қаласында тұрамын.`; console.log(info);
` (backtick) пайдаланып, ${} ішіне айнымалы жазыңыз!
🔧
Мәтін әдістері
let text = " JavaScript тамаша! "; console.log(text.length); // 20 - ұзындығы console.log(text.toUpperCase()); // бас әріп console.log(text.toLowerCase()); // кіші әріп console.log(text.trim()); // бос орындарды алып тастау console.log(text.includes("Java")); // true - қамтиды ма?

🔧 Мәтін құрастырғышы

Асем, 25 жаста, Алматыда тұрады

🎮 Мәтін әдістері көрмесі

length
Мәтін ұзындығын анықтау
"Сәлем".length → 5
toUpperCase()
Бас әріпке түрлендіру
"сәлем" → "СӘЛЕМ"
toLowerCase()
Кіші әріпке түрлендіру
"СӘЛЕМ" → "сәлем"
trim()
Бос орындарды алып тастау
" мәтін " → "мәтін"
includes()
Мәтін қамтиды ма?
"сәлем".includes("лем") → true
slice()
Бөлшек алу
"Алматы".slice(0,3) → "Алм"
replace()
Ауыстыру
"кіші".replace("і","ы") → "кыші"
split()
Бөлу (массивке)
"а,б,в".split(",") → ["а","б","в"]
let text = " JavaScript тамаша! "; console.log("Мәтін:", text); console.log("Ұзындығы:", text.length); // Әдісті таңдаңыз!
