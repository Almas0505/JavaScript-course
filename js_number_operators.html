Арттыру және азайту операторлары

++ Арттыру операторы

let count = 5; // Алдын ала арттыру (pre-increment) console.log(++count); // 6 (алдымен арттырып, содан кейін қайтарады) console.log(count); // 6 // Кейін арттыру (post-increment) let num = 5; console.log(num++); // 5 (алдымен қайтарып, содан кейін арттырады) console.log(num); // 6
Маңызды айырмашылық:
++x - алдымен арттырады, содан кейін мәнді қайтарады
x++ - алдымен мәнді қайтарады, содан кейін арттырады

-- Азайту операторы

let score = 10; // Алдын ала азайту (pre-decrement) console.log(--score); // 9 (алдымен азайтып, содан кейін қайтарады) console.log(score); // 9 // Кейін азайту (post-decrement) let lives = 3; console.log(lives--); // 3 (алдымен қайтарып, содан кейін азайтады) console.log(lives); // 2
Ойын мысалы:
Ойыншының жанын азайту кезінде lives-- жиі қолданылады

Құрама берілістер операторлары

let money = 100; // Қысқа түрде жазу тәсілдері: money += 50; // money = money + 50; → 150 money -= 20; // money = money - 20; → 130 money *= 2; // money = money * 2; → 260 money /= 4; // money = money / 4; → 65 money %= 10; // money = money % 10; → 5 money **= 2; // money = money ** 2; → 25 console.log(money); // 25
Кеңес:
Құрама операторлар кодты қысқартады және оқуды жеңілдетеді
