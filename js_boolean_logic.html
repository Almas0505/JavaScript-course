✅❌ Boolean және логикалық операторлар

Шындық пен жалғандықтың JavaScript әлемі

true

Шындық

✅ Шарт орындалған
✅ Мән бар
✅ "Иә" деген жауап
✅ Оң нәтиже

Boolean(1) → true
Boolean("text") → true
Boolean([]) → true
VS
false

Жалғандық

❌ Шарт орындалмаған
❌ Мән жоқ
❌ "Жоқ" деген жауап
❌ Теріс нәтиже

Boolean(0) → false
Boolean("") → false
Boolean(null) → false
&&
AND (ЖӘНЕ)
Барлық шарттар дұрыс болуы керек
true && true → true
true && false → false
||
OR (НЕМЕСЕ)
Кемінде бір шарт дұрыс болуы керек
true || false → true
false || false → false
!
NOT (ЕМЕС)
Мәнді кері бұрады
!true → false
!false → true
