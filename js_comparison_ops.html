⚖️ Салыстыру операторлары

JavaScript-тегі шарттар мен салыстырулар

==

Жұмсақ салыстыру

Түрлендіру жасап салыстырады
"5" == 5 → true

VS
===

Қатаң салыстыру

Түрін де тексереді
"5" === 5 → false

📊 Барлық салыстыру операторлары

==
Тең
5 == "5" → true
===
Қатаң тең
5 === "5" → false
!=
Тең емес
5 != "6" → true
!==
Қатаң тең емес
5 !== "5" → true
>
Үлкен
10 > 5 → true
<
Кіші
3 < 7 → true
>=
Үлкен/тең
5 >= 5 → true
<=
Кіші/тең
4 <= 9 → true
let a = 5; let b = "5"; let c = 10; console.log(a == b); // true - мән бірдей console.log(a === b); // false - түрі басқа console.log(a != c); // true - мәндер басқа console.log(a !== b); // true - түрі басқа console.log(c > a); // true - 10 > 5 console.log(a < c); // true - 5 < 10 console.log(a >= 5); // true - 5 >= 5 console.log(c <= 20); // true - 10 <= 20

🧪 Интерактивті салыстырғыш

→
5 == 5 = true

📋 Шындық кестесі

Мән 1 Мән 2 == === != !==
5 "5" true false false true
0 false true false false true
null undefined true false false true
"" 0 true false false true
true 1 true false false true
[] "" true false false true
⚠️ Маңызды ескерту: Жұмсақ салыстыру (==) кезінде JavaScript автоматты түрлендіру жасайды, бұл күтпеген нәтижелерге әкелуі мүмкін. Әрқашан қатаң салыстыру (===) пайдаланыңыз!

💡 Практикалық кеңестер

🎯 Қашан қай операторды пайдалану керек?

=== - Барлық жағдайда (ұсынылады)

== - Тек null/undefined тексеру үшін

>, <, >=, <= - Сандар мен мәтіндерді салыстыру үшін

// ✅ Дұрыс тәсіл - қатаң салыстыру if (age === 18) { console.log("Сіз кәмелетке толдыңыз!"); } // ✅ null/undefined тексеру үшін if (value == null) { // null немесе undefined console.log("Мән жоқ"); } // ✅ Сандарды салыстыру if (score >= 90) { console.log("Өте жақсы!"); } // ❌ Жұмсақ салыстыруды болдырмау if (input == true) { // "1", 1, "true" - барлығы true болады // Күтпеген нәтижелер! }
